PCHelper
========================

This is a nw.js UI Framework demo.

Feature
--------
- Custom Cursor Icon
- Element Animation
- Screen Switching Animation
- Opacity


Supported Controls:
------
- Frame
- Label
- Plane
- Image
- List
- Polygon
- TextBox
- Screen

Usage
-----
run ``nw .`` in the project directory
You can get ``nw`` from this repo https://github.com/nwjs/nw.js
